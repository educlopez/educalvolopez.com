Este caso de estudio se centra en el proyecto de desarrollo realizado para Sms Net2phone, en colaboración con la empresa Ringsouth. Incluyó la creación de una landing de marketing para promocionar el producto, así como el diseño y desarrollo de un dashboard utilizando Laravel y Bootstrap para la gestión de envío de SMS. Tras la transición de Ringsouth a Net2phone, se realizó una modificación de la interfaz de usuario para reflejar los colores y el logo de la nueva marca. Además, se crearon videos tutoriales explicativos para cada tipo de envío de SMS, incluyendo métodos con campos dinámicos y generación a partir de archivos .csv.

## Contexto

Sms Net2phone, una iniciativa de Ringsouth que evolucionó a Net2phone, buscaba mejorar su presencia en el mercado y brindar a sus clientes una solución eficiente y fácil de usar para el envío de SMS. Esto incluía la promoción del producto a través de una landing page atractiva y la creación de un dashboard intuitivo que permitiera a los usuarios gestionar sus envíos de SMS de manera eficaz.

## Creación de Landing de Marketing

La primera fase del proyecto consistió en la creación de una landing page de marketing para promocionar el servicio de envío de SMS de Sms Net2phone. Se diseñó una página atractiva y convincente que resaltaba las características y ventajas clave del producto. La landing se construyó utilizando HTML, CSS y elementos visuales impactantes, además de llamadas a la acción prominentes para impulsar la conversión de los visitantes.

## Desarrollo del Dashboard en Laravel y Bootstrap

Para la creación del dashboard de gestión de envío de SMS, se optó por utilizar el framework Laravel junto con el framework de diseño Bootstrap. Esto permitió una implementación eficiente y una interfaz de usuario receptiva.

Características Clave del Dashboard:

1. Gestión de Envíos: Se implementó una interfaz que permitía a los usuarios crear, programar y gestionar envíos de SMS. Los usuarios podían seleccionar destinatarios, redactar mensajes y programar el envío según sus necesidades.

2. Tipos de Envío Diversificados: Dado que había varios métodos de envío disponibles, se crearon pestañas y secciones separadas para cada tipo. Esto incluía envíos con campos dinámicos y envíos generados desde archivos .csv.

3. Personalización de Mensajes: Se ofrecía la capacidad de personalizar mensajes con campos dinámicos, lo que permitía un enfoque más individualizado en la comunicación.

4. Interfaz Intuitiva: Se priorizó una interfaz de usuario intuitiva con paneles organizados y opciones de navegación claras, lo que facilitaba la gestión de envíos incluso para usuarios nuevos.

## Adaptación a la Marca Net2phone

Con el cambio de Ringsouth a Net2phone, fue esencial reflejar la nueva identidad de marca en el dashboard. Se realizaron cambios en la paleta de colores y se incorporó el logo y otros elementos de diseño de Net2phone para mantener la coherencia con la marca.

## Creación de Videos Tutoriales

Se reconoció la necesidad de proporcionar orientación a los usuarios para aprovechar al máximo las características del dashboard. Se crearon videos tutoriales explicativos para cada tipo de envío de SMS. Estos videos ofrecían una guía paso a paso para que los usuarios comprendieran cómo utilizar eficazmente las diversas funciones y opciones.
